# JavaLearn
 ## 基础概念
 JDK（JAVA Develop Kit）java开发工具， 其中包括JVM--运行java的虚拟机，数据包--各种内置的库，开发工具--包括编译、运行、调试等工具<br>
 JRE（JAVA Run Environment）java运行环境，包括JVM 数据包和某些开发工具<br>
 ### 数据类型
long 和 float类型的定义 数据值后面需要加L/F  long a = 100L;  float b = 10.1F;<br>
